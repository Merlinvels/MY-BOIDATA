DIPLOMA PROJECT:

GESTURE BASED DC MOTOR SPEED CONTROL

This project is developed for the user to control the operation of a DC motor using Microcontroller PIC16F73.
This project uses 4 IR sensor for the operation or DC motor. These four sensor are responsible for start,stop ,speed up and speed down. According to the IR sensor output microcontroller generates the control signal. The microcontroller controls the driver section for the corresponding operation of the motor and thus the motor runs accordingly. The gesture recognition is achieved using IR sensor.
The user can control the speed of the DC motor in an efficient manner. The microcontroller program is written in assembly language.
